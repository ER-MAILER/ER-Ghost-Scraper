Installation Guide

System Requirements

  • Windows: Windows 10 or later (64-bit)
  • RAM: 4GB minimum, 8GB recommended
  • Storage: 500MB free space
  • Internet: Required for activation and updates

Installation Steps

  1. Download the software ZIP file from your email
  2. Extract the ZIP file using password: ghost11-02-2025 (Standard) or ghostpro11-02-2025 (Pro)
  3. Run the installer file (Windows: .exe)
  4. Follow the installation wizard instructions
  5. Launch the application and enter your license key
Note: Make sure to disable any antivirus software temporarily during installation if it blocks the installation.

Getting Started

Initial Setup

After installation and activation, follow these steps:

  1. Open ER Ghost Scraper from your desktop or applications folder
  2. Select your data extraction preferences
  3. Choose output format (CSV, Excel)
  4. Configure data filters if needed
  5. Start the extraction process

Interface Overview

The main interface consists of:

  • Dashboard: Overview of available data sources
  • Extraction Panel: Configure data extraction parameters
  • Results Viewer: Preview extracted data
  • Export Options: Choose output format and location

Troubleshooting

Common Issues

Installation Problems

  • Issue: Installer won't run
  • Solution: Right-click installer and "Run as Administrator" (Windows) or check Security & Privacy settings (macOS)

Activation Issues

  • Issue: License key not working
  • Solution: Ensure you're connected to internet and copy-paste the key exactly as provided

Performance Issues

  • Issue: Software running slowly
  • Solution: Close other applications and ensure sufficient RAM is available

Frequently Asked Questions

General Questions

Q: How often is the database updated?
A: Our Russian server database is updated regularly, typically weekly.

Q: Can I use the software on multiple computers?
A: Each license is for single computer use only.

Q: What kind of data is available?
A: The database contains various business and contact information collected from multiple sources.

Technical Questions

Q: The software won't start after installation
A: Try reinstalling and ensure all system requirements are met.

Q: Export function not working
A: Check if you have write permissions to the selected export folder.
